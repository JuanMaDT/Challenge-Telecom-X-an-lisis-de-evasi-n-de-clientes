# Challenge-Telecom X analisis de evasion de clientes
Data Science - ORACLE ONE 
Este proyecto tiene como objetivo analizar la evasión de clientes en la empresa Telecom X mediante técnicas de Análisis de Datos. A partir de datos reales de clientes, se busca identificar patrones de comportamiento y factores que influyen en la cancelación del servicio, con el fin de proponer estrategias de retención basadas en datos.
